O que fazer?

  1. Criar um arquivo HTML com nome index.html
  2. Insira os comandos básicos para iniciar o trabalho.
  3. Criar um currículo contendo
    1. Nome completo e apresentação (fale um pouco sobre vocês. Exemplo
    2. Fale sobre sua formação (Curso, período, instituição, etc)
    3. Apresente as linguagens de programação que já conhecem.
    4. Cite os idiomas que fala. (Ordene por proficiência)
    5. Adicione as formas de contato
      • Github
      • Linkedin
      • E-mail
  4. Adicione sua foto
  5. Adicione um índice com hyperlinks internos (direcionando o navegante para as seções corretas).
  6. Como enviar
    • Adicionar a página a um repositório no github e publicar o link no Ulife
